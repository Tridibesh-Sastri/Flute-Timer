# UI Modes Breakdown

## Mode 1 — Floating Mode
- **Scope**: Live practice visual bindings.
- **Layout**: Small overlay, minimal interface footprint.
- **Components**: Live session timer readout, live audio NoteEvent timer readout, minimal trigger toggles.

## Mode 2 — Dashboard Mode
- **Scope**: Historical analytics review platform and learning configuration.
- **Layout**: Full application scale UI logic.
- **Components**: 
  - Analytics metrics mappings, calendar mappings, deep dive lists.
  - Learning Timer control panel.
  - Alarm setting configuration module.
