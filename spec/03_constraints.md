# Strict Constraints

## Technology Constraints
- No frameworks (no React, Vue, etc.)
- Use only vanilla JS
- Single unified application tree
- Extremely lightweight processing
- Fast startup (< 1s)

## Storage Strategy
- System MUST strictly write structured data objects (Sessions + Notes).
- System NEVER stores or persists raw microphone audio.
- The structured schema must natively sustain high-level queries specifically fueling the future analytics engine mappings.
