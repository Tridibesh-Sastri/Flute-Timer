# flute-timer
A lightweight Electron desktop app that detects flute playing via microphone, automatically starts a timer when sound is heard, and stops when silence is detected for 300ms.

## Instructions for AI Agent
Read files in this order to fully understand and build the application:
1. `spec/*` -> the core features and constraints
2. `rules/*` -> strict coding and design guidelines
3. `skills/*` -> how to implement specific technical features
4. `tasks/*` -> step-by-step roadmap to build the app
5. `app/*` -> source code directory (you will write implementation here)
