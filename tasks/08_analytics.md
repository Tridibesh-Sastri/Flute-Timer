# 08 Analytics (Phase 4.1 Active)

## Current Scope
1. Render analytics only in Dashboard Summary tab.
2. Keep metrics read-only (no session mutation).
3. Use pure CSS bar charts only.

## Required Metrics
1. Session duration trend (recent sessions)
2. Note count per session
3. Practice frequency by weekday
4. Longest note
5. Shortest note
6. Practice streak

## Constraints
1. No external chart libraries.
2. Derive values from localStorage `sessions` only.
3. Dashboard rendering must not affect floating widget loops.
